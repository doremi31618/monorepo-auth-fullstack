# Product Specification: Authentication (R0)

> **Owner**: PM
> **Feature**: Authentication & User System (Pre-Monorepo)
> **Status**: Completed

## 1. Overview
Basic email/password authentication and Google OAuth integration using NestJS and SvelteKit.

## 2. Features
- **Sign Up**: Email/Password registration.
- **Login**: Email/Password login.
- **Google OAuth**: Social login.
- **Session Management**: JWT/Cookie based session.
- **Password Reset**: Email link flow.

## 3. UI Flows
(Migrated from R0 Documentation)
- Login Page -> Dashboard
- Signup Page -> Dashboard
- Forgot Password -> Email Sent -> Reset Page
